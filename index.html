BH-Backer--Originally-Blackhacker-Backer- by DcSlick

Shell Script for Jailbroken iPhone Users to backup user specified directories.

A simple backup script to aid in automatic or manual backups of jailbroken iPhones or other iDevices.

Dependencies

none

Install

Download to your phone, use either iFile to make it executable or chmod +x, execute from iFile, mobile terminal, or Putty.
